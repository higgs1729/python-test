# データ分析試験 模擬問題集アプリ

Python3エンジニア認定 **データ分析試験** 対策用の、ローカルで動く自習アプリです。
インストール不要。ブラウザで `index.html` を開くだけで動きます。

## 使い方
1. `index.html` をダブルクリック（既定のブラウザで開きます）
2. 出題する分野・出題数を選んで「試験を開始」
3. 4択から選ぶと即座に正誤・解説を表示
4. 最後に正答率（合格ライン70%）と分野別成績を表示

## 機能
- **分野別出題** … 9分野から任意に選択
- **本番形式** … 40問モードあり（合格ライン70%）
- **誤答復習** … 間違えた問題だけを再出題
- **成績の自動保存** … ブラウザの localStorage に累計成績を記録
- **解説付き** … 全問に解説あり

## ファイル構成
| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（画面・ロジック） |
| `questions.js` | 問題データ（**ここを編集すれば問題を追加・修正できます**） |
| `checklist.md` | 出題範囲の学習進捗チェックリスト |

## 問題の追加方法
`questions.js` の `window.QUESTIONS` 配列に、次の形式で追記します。
```js
{
  id: 46,               // 一意の番号
  cat: "pandas",        // 分野キー(下表)
  q: "問題文",
  choices: ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  answer: 0,            // 正解の選択肢インデックス(0〜3)
  exp: "解説文"
}
```
分野キー: `role / env / pybase / math / numpy / pandas / plot / ml / data`
