# データ分析試験 模擬問題集アプリ

Python3エンジニア認定 **データ分析試験** 対策用の、ローカルで動く自習アプリです。
インストール不要。ブラウザで `index.html` を開くだけで動きます。

## 使い方

### 起動方法: `start.bat` をダブルクリック
- ブラウザが自動で開き、サーバも起動します。終了は黒い画面で `Ctrl+C`。

### 簡易: ファイルを直接開く
- `index.html` をダブルクリックしても動きます。
- ただしこの場合はブラウザの localStorage に保存されます（フォルダ内 `progress.json` には保存されません）。
  ブラウザの履歴削除で消える点に注意。フォルダに残したい場合はサーバ起動を推奨します。

## 問題の追加方法
`questions.js` の `window.QUESTIONS` 配列に、次の形式で追記します。
```js
{
  id: 46,               // 一意の番号
  cat: "pandas",        // 分野キー(下表)
  q: "問題文",
  choices: ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  answer: 0,            // 正解の選択肢インデックス(0〜3)
  exp: "解説文"
}
```
分野キー: `role / env / pybase / math / numpy / pandas / plot / ml / data`
