# データ分析試験 模擬問題集アプリ

Python3エンジニア認定 **データ分析試験** 対策用の、ローカルで動く自習アプリです。

同じ `index.html`（UI・問題）を、以下の3通りの方法で起動できます。学習データ（成績・マーク）はいずれも
`progress.json` に保存され、環境をまたいで共有できます（ブラウザ直開き時のみ localStorage）。

## 使い方

### A. デスクトップアプリ（推奨・配布可）
ネイティブウィンドウで起動します。ブラウザもポートも不要。

- **開発時に動かす**：`pip install -r requirements.txt` の後、`python app.py`
- **配布用の単一exeを作る**：`build.bat` をダブルクリック → `dist\データ分析試験.exe` が生成されます。
  この exe は **Python 未インストールのPCでもそのまま動作**します（Windows 10/11 標準の Edge WebView2 を使用）。
  学習データ `progress.json` は **exe と同じフォルダ**に保存されます。
- 終了は画面右上の「⏻ 終了」ボタン、またはウィンドウを閉じるだけ。

### B. ブラウザ運用（テスト用）: `start.vbs` をダブルクリック
- `server.py` が起動し、既定ブラウザで開きます。成績はフォルダ内 `progress.json` に保存。
- 終了は画面右上の「⏻ 終了」ボタン（サーバも停止します）。

### C. 簡易: `index.html` を直接ダブルクリック
- サーバなしで動きますが、保存先はブラウザの localStorage です（`progress.json` には保存されません）。

## 問題の追加方法
`questions.js` の `window.QUESTIONS` 配列に、次の形式で追記します。
```js
{
  id: 46,               // 一意の番号
  cat: "pandas",        // 分野キー(下表)
  q: "問題文",
  choices: ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  answer: 0,            // 正解の選択肢インデックス(0〜3)
  exp: "解説文"
}
```
分野キー: `role / env / pybase / math / numpy / pandas / plot / ml / data`
